# PROTOTYPE — 单机 Web MVP 端到端交互

这个目录只用于回答 Wayfinder 票据“原型确认单机 Web MVP 的端到端交互”，不是正式产品代码。

三个结构不同的方案放在同一页面，通过 `?variant=A|B|C` 和底部切换器比较：

- A — 阶段导轨
- B — 任务台账
- C — 实例优先

## 运行

```bash
python3 -m http.server 4173 --directory prototype-web-mvp
```

打开 <http://127.0.0.1:4173/?variant=A>。

## 建议评议路径

1. 在“脚本管理”中测试复制与下载反馈。
2. 进入“报告生成”，切换粘贴与上传；用“查看错误示例”检查结构化错误。
3. 点击“分析并生成报告”，切换实例并编辑 Markdown。
4. 测试即时安全预览、权威校验、当前 DOCX、全部 ZIP 和结束会话。
5. 用底部左右按钮或键盘方向键比较三种方案；输入框聚焦时方向键不会切换方案。
